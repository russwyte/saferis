# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | Yes                |

## Reporting a Vulnerability

If you discover a security vulnerability in saferis, please report it responsibly:

1. **Do not open a public issue.**
2. Use [GitHub's private security advisory feature](https://github.com/russwyte/saferis/security/advisories/new) to report the vulnerability.
3. Alternatively, email 356303+russwyte@users.noreply.github.com with details.

You should receive a response within 7 days. We will work with you to understand the issue and coordinate a fix before any public disclosure.
